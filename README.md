# Leptos Weird SSR and interval stuff

```
cargo leptos serve
```

1. Go to [localhost:3050](http://localhost:3050)
2. Click on `"Go Away"`
3. Wait 10 seconds (the interval will be visible)
4. After you're done reload the page (the interval will NOT be visible)
